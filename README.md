# 🚀 Gestor de Comercio — v1.0

> 🖥️ Software de escritorio para gestión de inventario, ventas, cajas y usuarios.

---

## 🎊 ¿Qué es?

Un sistema de gestión integral para **pequeños y medianos comercios**. Controlá tu inventario, procesá ventas, administrá usuarios con permisos granulares, y mirá reportes diarios — todo desde una interfaz moderna, rápida y hermosa.

**Pensado para:** verdulerías, restaurantes, almacenes, carnicerías, panaderías, ferreterías, kioscos, farmacias, librerías... y básicamente cualquier negocio que venda productos.

---

## ✨ Features

### 🎨 UI de nivel profesional
- **Glassmorphism** con transparencias, blur y animaciones
- **Modo oscuro y claro** con toggle animado (sol/luna, estrellas, nubes)
- **Sidebar** con avatares circulares, íconos customizables vía PNG
- **Modales** con animaciones de entrada y salida
- **Grilla visual de productos** para el flujo de venta
- **Carrito flotante** con badge y animación de "vuelo" al agregar
- **Dropdown de usuario** con historial de sesiones recientes

### 👥 Sistema de roles y permisos
- **Admin, Empleado, Cajero** y **Personalizado** con 10 permisos individuales
- Panel de permisos con toggles tipo Discord
- Cada usuario ve solo lo que necesita

### 📦 Inventario
- CRUD completo de productos
- **Por peso o por unidad**
- Búsqueda en tiempo real y alertas de stock bajo

### 💰 Ventas
- **Grilla visual** con placeholders de color
- **Popup de cantidad** al clickear un producto
- **Carrito flotante** con animación
- **Pago combinado** (efectivo + Mercado Pago en una sola venta)
- **Ticket imprimible** con todos los detalles
- **Filtros avanzados**: por fecha, método de pago, orden por monto

### 📊 Dashboard
- Resumen del día con cards
- Productos con stock bajo destacados
- Últimas ventas con acceso al detalle

### 🖼️ Personalización total
- Poné tus propios PNGs para iconos y avatares
- Selector de emojis en el setup inicial
- Subí fotos para el negocio y cada usuario
